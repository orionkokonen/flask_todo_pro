## 後で追加したい機能（候補）

- ドラッグ＆ドロップでカンバン移動（TODO → DOING → DONE を指で移動）
- タグ / 優先度
- フィルタ強化（期限が近い順 / プロジェクト別 / 自分の担当だけ / タグ別 /など）
- TODOだけ・DOINGだけ・DONEだけ・WISHだけの画面に切り替えられるようにする
- ショートカット（「＋」固定ボタン、クイック追加、テンプレ）
- ダークモード、レスポンシブ微調整、アクセシビリティ（フォーカス / キーボード操作）
- コメント / 履歴（監査ログ）（だれがいつ完了にしたか）
- RPG化（クエスト化 + レベル制 + 実績）
- リアルタイム共同編集（WebSocket）
- Googleログイン対応（OAuthログイン）
- オフライン・ファースト＋同期（衝突解決）
- バックグラウンドジョブ＋キュー（通知/集計/重い処理を分離）
- 観測性（Observability：ログ/メトリクス/トレース）
- 通知（締切○日前に通知：PWAのPush通知 or メール通知）
- 繰り返しタスク（毎週/毎日、スヌーズ）
- 権限ロール（チームのOwner/Admin/Member、プロジェクトごと権限）
- テーマ(壁紙の色)を青・緑・オレンジ・ピンク・水色など追加
- マイページ作る（・完了したタスクの数・未完了のタスクの数・今後7日間のタスク
  ）
- カレンダーページを作る。(カレンダーの日付をタップすると、その日締め切りのタスクが表示される)
- タスク削除時、「本当に消しますか？」と確認する2段階構成にする
- プロジェクトの欄で、プロジェクトをクリックしたときにそのプロジェクトのタスクが表示されて、かつタスクをタップすると詳細画面に飛ぶようにする
- REST API + OpenAPI/Swagger + Token認証
- 招待リンク / 招待メールでチーム参加
- ゴミ箱 / アーカイブ / 復元
- 添付ファイル・画像アップロード
- タスク依存関係（blocked by / blocks）
- 見積もり時間 / 実績時間の記録
- 複数選択の一括操作（移動 / 削除 / プロジェクト変更 / 期限変更）
- インポート / エクスポート（CSV / JSON / Markdown）
- 分析ダッシュボード（完了率 / 期限超過率 / リードタイム / プロジェクト別進捗）
